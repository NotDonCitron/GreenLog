Kalender-Feed Interaktion

Was passiert wenn man einen Tag auswählt?

A

Feed zeigt nur diesen Tag

Kalender filtert den Strain-Feed auf den ausgewählten Tag. Kein Tag gewählt = alle anzeigen.

B

Kalender dient nur zur Visualisierung

Dots zeigen an wann was gesammelt wurde. Klick auf Tag scrollt im Feed zum entsprechenden Eintrag (wie Instagram/YouTube Timeline).

C

Beides: Filter + Scroll-to

Tag wählen filtert den Feed. Nochmal klicken scrollt zum ersten Eintrag dieses Tags im Feed.

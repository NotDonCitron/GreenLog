<task type="auto">
  <name>Task 1: Create API endpoint for fetching pending members</name>
  <files>src/app/api/organizations/[organizationId]/pending-members/route.ts</files>
  <action>
Create GET endpoint to fetch pending membership requests for an organization.

**Route:** `GET /api/organizations/[organizationId]/pending-members`

**Logic:**
1. Authenticate user
2. Check if user is admin or gründer (use USER_ROLES.GRUENDER, USER_ROLES.ADMIN)
3. Fetch organization_members where:
   - organization_id matches
   - membership_status = 'pending'
4. Join with profiles to get user info (id, username, display_name, avatar_url)
5. Return array of pending members with profile data

**Response:**
```typescript
{
  data: {
    pendingMembers: Array<{
      id: string;
      user_id: string;
      joined_at: string | null;
      user: {
        id: string;
        username: string | null;
        display_name: string | null;
        avatar_url: string | null;
      }
    }>
  },
  error: null
}
```

**Error codes:**
- 401: Unauthorized
- 403: Not admin
- 404: Organization not found

3. Returns empty array when no pending members
</verify>

GET /api/organizations/[organizationId]/pending-members returns list of pending membership requests with user profile info.
</done>
